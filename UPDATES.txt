## 1. ПРОКСИ ДЛЯ РОТАЦИИ IP (Proxy Rotation)

✅ ЧТО БЫЛО ДОБАВЛЕНО:

A) Три новых файла:
   - proxy_scraper.py    → Скрапит бесплатные прокси с API и тестирует их
   - init_proxies.py     → Быстрая инициализация прокси список
   - proxies.json        → Кэш рабочих прокси (уже создан с 24 адресами)

B) Новые функции в config.py:
   - PROXIES = []                           → Список прокси для ротации
   - AUTO_FETCH_PROXIES = True              → Автоподгрузка свежих прокси на старт
   - PROXY_TEST_TIMEOUT = 60                → Таймаут для тестирования
   - SHOE_KEYWORDS = {...}                  → Ключевые слова обуви
   - CLOTHING_EXCLUDE = {...}               → Слова одежды для исключения

C) Функции в main.py:
   - get_next_proxy()                       → Ротирует прокси по очереди
   - Все HTTP запросы теперь используют прокси

D) Как это работает:
   1. При старте бота загружаются прокси из proxies.json
   2. Если AUTO_FETCH_PROXIES=True, бот пытается найти свежие прокси
   3. Если поиск не удаётся за 60 сек, использует кэшированные
   4. Прокси ротируются автоматически при каждом запросе

D) Где находятся прокси:
   - proxies.json       → Текущий список (24 бесплатных IP)
   - proxy_scraper.py   → Может обновить список вручную:
                         python proxy_scraper.py (медленно, тестирует каждый)

## КАК ИСПОЛЬЗОВАТЬ:

1. БОТ СТАРТУЕТ АВТОМАТИЧЕСКИ:
   python main.py
   
   → Загружает 24 прокси из proxies.json
   → Отправляет запросы через разные IP (~каждый 24-й запрос)
   → При карточке товара проверяет каждую картинку
   → Отклоняет одежду, принимает только обувь

2. ОБНОВИТЬ ПРОКСИ ВРУЧНУЮ:
   python proxy_scraper.py
   
   → Скрапит свежие прокси с сайтов
   → Тестирует каждый на скорость
   → Сохраняет топ рабочих в proxies.json
   ⚠️ МЕДЛЕННО - может занять 2-3 минуты

3. БЫСТРАЯ ИНИЦИАЛИЗАЦИЯ:
   python init_proxies.py
   
   → Сразу создает proxies.json с 24 прокси
   → Без тестирования, быстро

## ПРОВЕРКА/ОТЛАДКА:

1. Смотреть список текущих прокси:
   cat proxies.json

2. Проверить синтаксис:
   python -m py_compile main.py config.py proxy_scraper.py

3. Обновить список на старте:
   - Отредактируй config.py: AUTO_FETCH_PROXIES = True
   - Первый запуск будет медленнее (~90 сек)

4. Отключить прокси:
   - В config.py: PROXIES = []
   - Или: AUTO_FETCH_PROXIES = False
